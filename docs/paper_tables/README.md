# Paper Table Exports

This directory stores exported paper-preset summaries generated by:

```bash
python run_experiment.py --export-paper-tables
```

You can also export to another directory or filter by preset name:

```bash
python run_experiment.py --export-paper-tables some/output/dir
python run_experiment.py --export-paper-tables --filter table6
```

Artifacts:

- `index.json` / `index.tsv`: export index for all presets in the batch
- `<preset>.json`: machine-readable structured summary for one paper preset
- `<preset>.tsv`: flat tabular summary for one paper preset

Notes:

- `supported`, `partial`, and `unsupported` presets are all exported so the
  directory remains a complete paper-facing inventory.
- A preset export reflects the latest local artifacts currently present under
  `sart/result/`.
- Single-preset summaries can also be generated directly with
  `python run_experiment.py --summarize-paper-preset <preset>`.
- `table5_fallback_frequency` only contains Stage 1/2/3 fallback percentages
  after the underlying preset has been rerun with the newer `StageOutcome:`
  log markers.
